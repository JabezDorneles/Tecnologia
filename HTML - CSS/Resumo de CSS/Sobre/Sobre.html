CSS no HTML:

Com ordem de prioridade!

  1. Inline: usado como atributo de TAG, com função pontual.
    2. - Procure sempre evitá-la!
    EX: <h1 style="">

  2. Internal: usado dentro do HEAD, com função local.
    3. - Use para pequenas configurações!
    EX: <head><style>...</style></head>

  3. External: usado fora do HTML, com função global.
    4. - Use sempre que puder!
    Chamamos, assim: <head>link:css</head>
    ou <head><link rel="stylesheet href="..."></head>
