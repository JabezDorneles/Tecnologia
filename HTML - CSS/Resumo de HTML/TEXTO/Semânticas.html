SEMÂNTICAS

        
formatações

        <strong> (FORTE - NEGRITO)
        <em> EMPHASIS (ÊNFASE (ITÁLICO) - DAR SUMA IMPORTÂNCIA)
        <mark> MARCADOR (DAR FOCO)
        
        <small> pequeno (comentário - texto breve)

        <dell> DELETAR (RISCAR TEXTO - mostrar como excluído)
        <ins> INSERIR (SUBLINHAR TEXTO - mostrar como modificado)

        <sup> 12 (SUPERIOR - para necessidade semântica)
        <sub> H2 (INFERIOR - para necessidade semântica)

        <pre> (texto pré-formatado - inclui: tabulação, quebra de linha e espaço.)
        <code> (if (2==2) True) - para trechos de código

        <q> quotation (citação (curta - com aspas))
        <blockquote cite='site'> (bloco de citação (longa - cite = referência))

        <abbr title"NOME COMPLETO">SIGLA</abbr> (ABBR)

        <bdo dir='rtl'> (bdo = BiDirectional override (Bi=2; Direção; Substituição); dir = direction(direção); rtl = right-to-left)

        <address> (referência de endereço)

        <aside> (À PARTE - Algo a parte com o conteúdo, mas indo pro outro assunto)

        
Listas

        
Simples:

        <ol> (Ordered List (Lista Ordenada) - lista em ordem)
            <ol type='1,A,a,I ou i' start='começar em que ordem? ex: 2'>

        <ul> (Unordered (Não Ordenada) - lista)
            <ul type='disc (padrão), circle ou square'>

        <li> (List Item (Item da lista) - cada item)

        
Composto:

        <ol> (type padrão: 1...)
            <li>Item 1</li>
            <ol type='a'>
                <li>Item 1.1</li>
            </ol>
        </ol>

        
de Definição: (util para glossário)

        <dl> (Definition List (Lista de Definição) - Tipo de lista de definição)
        <dt> (Definition Term (Termo de Definição) - Representação do conceito)
        <dd> (Definition Description (Descrição da Definição) - Conceito da representação)
