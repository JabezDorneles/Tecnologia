Imagens de acordo com a largura da janela

Tamanhos para imagens:

  • P - png: 300X300
  • M - png: 700X700
  • G - JPG (para fotografia): 1000x1000

Codigo de exemplo:

        
            <picture> Esta é a tag para escolher uma das imagens abaixo, começando de cima para baixo.
                <source media="(max-width: 750)" srcset="foto-P.png type="image/png"> Largura menor que 750 pegará está imagem
                <source media="(max-width: 1050)" srcset="foto-M.png" type="image/png"</source> Largura menor que 1050 pegará está imagem
                <img src="foto-G.jpg" alt="Foto de... - tamanho: G (principal)"> Se nenhuma, está imagem.
            </picture>

Audios de acordo com a compatibilidade de navegador

Formatos de Audio (principais):

  • mp3: audio/mpeg
  • ogg: audio/ogg
  • wav: image/wav

Atributos da Tag Audio:

  • src (ou no source)
  • controls (controlador de audio)
  • autoplay (começa automaticamente)
  • loop (repete sem fim)
  • preloaod (pré-carga)
    • metadata (informações: tamanho, tempo...)
    • auto (carrega automaticamente)
    • none (não carrega automaticamente)

Código de exemplo:

        
            <audio preload="metadata" controls autoplay loop> Aqui configura o audio, podendo até mesmo colocar por aqui mesmo o mp3.
            O audio a ser escolhido no source começa de cima para baixo.
                <source src="audio.mp3" type="audio/mpeg"> MP3
                <source src="audio.ogg" type="audio/ogg"> ogg
                <source src="audio.wav" type="image/wav"> wav
            </audio>

Videos de acordo com a compatibilidade de navegador

ogv(video/ogg) é menos popular

Conversor:

Handbrake (programa)

Formato de + leve para mais pesado (por teste):

Os 2 primeiros são os principais
  1. m4v: video/mp4
  2. webm: video/webm
  3. mp4: video/mp4
  4. ogv: video/ogg

Atributos da Tag Video

  • src (ou no source)
  • width (largura)
  • poster (thumb)
  • controls (controlador de video
  • autoplay (play automático)
  • loop (repetidor infinito)
  • preload (pré-carga)
    • metadata (informações: tamanho, tempo...)
    • auto (carrregar automaticamente)
    • none (não carregar automaticamente)
        
            <video width="600px" poster="thumb" preload="none" controls autoplay loop>
                <source src="video.m4v" type="video/mp3">
                <source src="video.webm" type="video/webm">
                <source src="video.mp4" type="video/mp4">
                <source src="video.ogv" type="video/ogg">
            </video>
