Ligações

<a> Anchor (âncora)
serve para dar apoio ao hipertexto, fazendo uma ligação.

É preciso dar a referência do lugar

Atribuindo a ela...

href="https://www.exemplo.com" HyperText Reference (Referência de hipertexto)
o lugar que a âncora dará apoio cria um link para poder acessá-la.

Quando o link do site possui versôes diferentes de idioma e/ou país, podemos optar em especificar a versão principal com:

hreflag='pt-br' HyperText Reference Language (Linguagem de referência de hipertexto)
versão principal do link.

O alvo é o lugar do hipertexto do link; o tipo de alvo por padrão é entrá-la pela mesma aba que esteja a sua âncora (o apoio).

target='_self(padrão)' Alvo
tipo de alvo
  • target=""
      • _blank
      nova janela
      _self
      atual
      _top
      desfaz os frames e abre no navegador completo
      _parent
      similar ao de cima; só que com uma janela mãe mais próxima
      -nome-do-frame
      indica algum frame, se tiver.

Para dizer para onde o site vai levar, temos o:

rel="" relationship (relação)
tipo de relação que tem com o acessamento do link
  • rel=""
      • next Proximo
      proxima página
      prev Previous(anterior)
      página anterior
      author Autor
      ator do artigo
      external Externo
      outro site
      nofollow Não Seguir
      propaganda ou não tem o aval(garantir que o site é isso ou aquilo)
      dofollow Seguir
      autoridade

E se é para baixar algum arquivo, devemos atribuir o:

download="" Baixar
Indica que é pra baixar
o valor vazio deixa com o nome do arquivo, ou vc pode optar em escolher algum nome.
